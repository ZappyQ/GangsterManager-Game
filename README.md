# GangsterManager

**GangsterManager** is a deep, text-based strategic life simulator that puts you in the heart of the criminal underworld. Set in a gritty urban landscape, the game challenges you to rise from a small-time hustler to a legendary crime boss through strategy, resource management, and ruthless ambition.
